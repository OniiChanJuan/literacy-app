Notes for this mockup

  • Smart-hide action bar. Shows when scrolling up, hides when scrolling down. Implementation will use JS scroll-direction detection. This is the "partially sticky" pattern you wanted.
  • Bottom tab nav stays visible. When the action bar shows, both action bar and tab nav are on screen (top of action bar above tab nav). When user scrolls down, both hide together.
  • CrossShelf Score gets a teal-tinted card treatment. Visually distinguishes from contributing scores below. The headline status.
  • Franchise/series strip below the score, before contributing pills. Shows "Part of: [Franchise Name] · Position in series · Year." Small franchise thumbnail on the left, chevron on the right hints at tap-to-franchise-page. Hidden entirely if the item isn't part of any franchise (Babel is standalone — no strip). Works for movies (MCU, LOTR), books (Stormlight Archive, Discworld), games (Mass Effect trilogy), manga (long-running series), etc.
  • "Blended from 3 sources + community" sub-label tells the user the score isn't fabricated — it's transparent about composition.
  • Contributing scores capped at 3 with "+ N more" pill. The pill itself is styled distinctly (teal tint, border) so it reads as an action, not a value. Tap to expand the full list.
  • Rating distribution shows the 47 community ratings. Resurrected aggregate-score.tsx UI per the design decisions. Hidden entirely when an item has fewer than 10 ratings — empty bars would be misleading.
  • Your activity uses the same muted gold treatment as the serendipity slot on For You — consistent visual language for "this is about you specifically." Hidden when the user hasn't engaged; replaced by a Rate prompt card.
  • People row, where-to-watch, cross-shelf, reviews, related all follow established editorial patterns from For You. Each is hidden entirely if the item lacks data for that section — no empty CTAs.
  • "Cross your shelf" on detail page is the inverse: instead of "because you rated X highly," this is "if you love this, also try these." Same connections, different framing.
  • Empty-state rules: Rating distribution requires ≥10 ratings. Community pill requires ≥10 ratings. Recommend % pill requires ≥5 tags. Reviews require ≥1. Where to watch shown only when streaming data exists. Cross-shelf and Related hidden if no connections/related exist. Sparse items see a much shorter page reflecting what's actually known.
