What's different in this state

  • Red-dashed "hidden section" markers are diagnostic only — they show what's missing and why. They wouldn't appear on the production site. The user would just see the page flowing past sections that don't exist.
  • Score row uses neutral border instead of teal-tinted. Subtle signal: the score is from a single source (Goodreads), not a robust blend. The sub-label spells it out: "Based on Goodreads — community ratings coming as users review."
  • Contributing pills shows only Goodreads — that's the only score data we have. No fake Community pill or Recommend pill below threshold.
  • Rate prompt replaces Your activity since the user hasn't engaged. Dashed border + teal accent. Different visual treatment from the populated gold-bordered Your activity card.
  • Where to watch hidden — books don't have streaming platforms.
  • People shows just R.F. Kuang as Author. Row collapses naturally — no padding to fake fullness.
  • Cross your shelf still appears because the 133 hand-curated connections include Babel. Reuses cross-shelf data from Stage 4.
  • Franchise/series strip also hidden — Babel is a standalone book, not part of any series. The strip only appears when an item has a franchise/series association. For items that are part of one (MCU movies, LOTR trilogy, Stormlight Archive books, long-running manga, etc.) it appears between the score row and contributing pills.
  • Reviews and More like this hidden — no reviews yet, no related books cataloged yet.
  • Net effect: page is dramatically shorter than the LOTR detail page. Honest reflection of what's actually known about the item. As community ratings, reviews, and catalog expansion arrive, sections appear automatically.

Empty-state thresholds to bake in

  • Rating distribution: requires ≥10 community ratings
  • Community contributing pill: requires ≥10 ratings
  • Recommend % pill: requires ≥5 recommend tags
  • Reviews section: requires ≥1 review
  • Where to watch: shown only when streaming data exists for the item
  • Cross your shelf: hidden if zero connections
  • More like this: hidden if zero related items
  • Your activity card: hidden if user hasn't rated; replaced with Rate prompt
  • Score row treatment: teal-tinted (robust blend) vs neutral-bordered (thin inputs) based on number of contributing sources + community threshold
